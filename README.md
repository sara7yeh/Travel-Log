# 拍照灵感搭配库

移动端优先的本地网页原型，用来记录拍摄企划、参考图/视频、穿搭、姿势和适配地点类型。

## 微信小程序版

原生微信小程序代码位于 `miniprogram/`。打开微信开发者工具，选择“导入项目”，项目目录选择本仓库中的 `miniprogram` 文件夹即可。当前使用测试 AppID `touristappid`，适合本机预览；准备发布时，在微信公众平台注册小程序并把 `project.config.json` 中的 AppID 换成自己的。

小程序数据和上传的照片/视频保存在当前微信设备本地，不会与网页端或其他手机同步。

## 功能

- 拍摄企划与已拍作品分区
- 点击企划查看全部参考照片和视频
- 图片全屏查看、放大、缩小与重置
- 企划可标记为已拍并移动到已拍作品
- 已拍作品可上传自己的成片和视频
- 一个企划保存多张参考图和视频
- 具体想拍内容，例如 cosplay、马面裙写真、云南漂流视频
- 衣服/造型、姿势/构图、地点类型标签
- 可选具体地点和备注
- 想拍/已拍状态
- 按主题、状态、衣服、姿势、地点类型筛选
- 使用 IndexedDB 本机保存数据和图片
- 多状态原创小猫品牌形象：拍摄、计划、筛选、空状态与完成状态

## 公开网址

推送到 GitHub 后，GitHub Actions 会自动部署网页：

```text
https://sara7yeh.github.io/Travel-Log/
```

首次部署需要在仓库的 `Settings > Pages` 中，将 `Build and deployment` 的来源设置为 `GitHub Actions`。本地的 `127.0.0.1` 地址只能在自己的电脑上访问，不能发送给其他人。
