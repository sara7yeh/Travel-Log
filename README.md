# 拍照灵感搭配库

移动端优先的本地网页原型，用来记录拍摄企划、参考图/视频、穿搭、姿势和适配地点类型。

## 功能

- 拍摄企划与已拍作品分区
- 点击企划查看全部参考照片和视频
- 图片全屏查看、放大、缩小与重置
- 企划可标记为已拍并移动到已拍作品
- 已拍作品可上传自己的成片和视频
- 一个企划保存多张参考图和视频
- 具体想拍内容，例如 cosplay、马面裙写真、云南漂流视频
- 衣服/造型、姿势/构图、地点类型标签
- 可选具体地点和备注
- 想拍/已拍状态
- 按主题、状态、衣服、姿势、地点类型筛选
- 使用 IndexedDB 本机保存数据和图片
- 原创拍照小猫品牌形象与移动端适配界面
